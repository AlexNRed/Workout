# Workout

# Fitness Workout Tracker
A java program that models a fitness workout session with trainers

# classes
WorkOut: Main class that represents all the functions of a workout session
Trainer: Secondary class that represents the fitness trainer
App: the main tester class

# Description
What my program can do
Calculates the calories burned per minute and total calories burned
assigns a trainer to a workout
can extend the workout duration
Once the workout is done, it will be shown
